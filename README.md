# K's Loan Ledger

💼 個人向けの簡易貸付管理CLIアプリ
紙の借用書をもとに、貸付・返済・残高をCSVで記録・照会できます。

---

## 🔧 使用技術
- Python（標準ライブラリのみ使用）
- CSVファイル保存
- コマンドライン操作（CLI）

---

## 📦 現在の機能（2025年5月19日時点）
- 顧客ごとの貸付記録
- 返済記録と履歴の表示
- 残高照会（貸付-返済）
- 顧客IDの補正処理

---

## ▶ 使用方法（CLI版）

1. このプロジェクトをローカルに保存
2. ターミナルで以下を実行

```bash
python main.py
